### 配置163邮箱

```
以163邮箱为例说明。
SMTP 主机是 smtp.163.com。
SMTP 端口默认是25，使用 SSL 时端口为465/994，使用 TLS 时端口为587。
SMTP 账号密码是登录邮箱的账号密码或账号授权码。
配置邮件服务器与邮件发送后，可添加测试收件人并点击测试连接按钮，如果配置正确，页面出现提示信息，同时 JumpServer 会发送一条测试邮件到 SMTP 账号邮箱。
```
