# centos7作为客户端连接Socks5上网



## 安装软件

```
yum install -y curl yum-utils
yum install -y epel-release
yum install -y proxychains-ng

使用 yum 命令安装 proxychains-ng，这是一个允许应用程序通过 SOCKS5 代理服务器连接到网络的工具。
yum install -y proxychains-ng
```



## 修改配置文件

```
vi  /etc/proxychains.conf
socks5  192.168.67.78   1080    lamer   secret

```



## 测试

```
proxychains curl https://www.baidu.com?ssssssssssss

```
